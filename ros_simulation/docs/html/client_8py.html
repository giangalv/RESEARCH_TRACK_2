Detailed Description

Provides a ROS client that subscribes to odometry information, publishes position and velocity information, sends goals to the action server, and listens for user input to set the goal coordinates.

Author
Galvagni Gianluca
Version
0.1
Date
14/03/2023

Subscribes to:

  • '/odom': Topic where there are the real robot coordinates.

Publishes to:

  • '/msg/Pos': Where write the (x,y) coordinates and the velocities (vx and vy).
  • '/goal_topic' (geometry_msgs/Point) : User's desired position.
