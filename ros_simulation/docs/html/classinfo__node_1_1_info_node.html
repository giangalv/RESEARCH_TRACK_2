Detailed Description

ROS node to the robots' information (x,y,vx and vy) and prints the distance between robot and target.

This class subscribes to odometry information, sends goals to the action server and listens for user input to set the goal coordinates.

Callback function for pos subscriber.

Parameters
dataA Pos message containing the current position and velocity information.

This function is the callback for the /pos subscriber. It calculates the Euclidean distance between the current position and the desired position, as well as the average velocity, and prints this information if enough time has passed since the last print.
