def client.Client.__init__ (   self)

Initializes the ROS client class.

This function initializes the ROS client class by setting two variables with null values, creating a publisher to publish position information, creating a subscriber to listen for odometry information, creating an action client for the reaching_goal, and waiting for the action server to come usable.

Listens for the user input to set the goal coordinates and sends this information.

This function listens for user input to set the goal coordinates, sends the goal to the action server and to the message, and cancels the goal if the user input is 'c'.
